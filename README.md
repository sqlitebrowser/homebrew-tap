# homebrew-tap

## Installation
```
brew tap sqlitebrowser/tap
```

## Casks
| Cask                             | Description                                                  |
| :------------------------------- | :----------------------------------------------------------- |
| db-browser-for-sqlite            | DB Browser for SQLite (Stable)                               |
| db-browser-for-sqlite-nightly    | DB Browser for SQLite (Nightly)                              |
| db-browser-for-sqlcipher-nightly | DB Browser for SQLCipher (Nightly)                           |
| db-browser-for-sqlite-[ver]      | A version of the `A.B.C` format is available in the `ABC` format.<br />For example: `db-browser-for-sqlite-3122` for 3.12.2 version |

## Formulae
| Formula                 | Description                          |
| :---------------------- | :----------------------------------- |
| db4sqt@5                | Qt 5.x                               |
| db4ssqlcipher           | SQLCipher (x86_64)                   |
| db4ssqlitefts@5         | SQLite 3                             |
| db4subopenssl@3         | OpenSSL (arm64) for build SQLCipher  |
| db4subopenssl@3-x86_64* | OpenSSL (x86_64) for build SQLCipher |
| db4subqt@5*             | Qt 5.x                               |
| db4subsqlcipher*        | SQLCipher                            |
| db4subsqlcipher-x86_64* | SQLCipher (x86_64)                   |
| db4subsqlitefts@5*      | SQLite 3                             |

> *: These formulae are require Rosetta 2

> You can install Rosetta 2 via `softwareupdate --install-rosetta` command.
